=======
Hermod
=======

Installation of Hermod.

This is tested on Ubuntu 12.04.4:

 sudo apt-get install build-essential git-svn bison flex gfortran libatlas-base-dev python-dev pkg-config cmake python-virtualenv python-zc.buildout
 git svn clone -s -r1299:HEAD http://svn.rss.chalmers.se/svn/odinsmr/hermod
 cd hermod
 python virtualenv.py --system-site-packages .
 buildout2.7 bootstrap
 bin/buildout
