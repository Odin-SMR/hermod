=======
Hermod
=======

Installation of Hermod.

 sudo apt-get install build-essential git-svn bison flex gfortran libatlas-base-dev python-dev pkg-config cmake
 git svn clone -s -r1299:HEAD http://svn.rss.chalmers.se/svn/odinsmr/hermod
 cd hermod
 python virtualenv.py .
 bin/python bootstrap.py
 bin/buildout
